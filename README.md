## What This Mod Changes

| Feature || Vanilla | EasierPower |
|---|---|---|---|
| Overclock power multiplier (250% clock) || up to ~3.36x | up to 2.0x |
| Somersloop power scaling || boost² (up to 4x) | boost^1.618 (up to ~3.07x) |
| Variable-power building cycle || Complex, unpredictable | Simplified, predictable |

This mod is purely a power-balance tweak — no new content is added.
